# Robot-Logic-Playground
## Overview
Robot Logic Playground is a fun, interactive single-page web app (SPA) for learning basic programming and logical thinking by guiding a robot through mazes. Drag and drop commands to move the robot, avoid walls, and reach the goal!  

This project is perfect for students and beginners who want to learn programming logic in a visual and playful way.  

---

## Features
- **Home Page**
  - Floating 3D robot mascot
  - App title and subtitle
  - “Enter Playground” button to go to levels

- **Levels Page**
  - Predefined levels (Level 1, 2, 3…)
  - Custom map builder option
  - Back to Home navigation

- **Builder / Program Page**
  - Flexbox layout: Toolbox, Program Area, Map Preview
  - Scratch-style commands: Move Forward, Turn Left/Right, Repeat ×N
  - Add commands by click
  - Custom map editor for walls, start, goal

- **Simulation Page**
  - Robot executes program step-by-step with animations
  - Controls: Start, Pause, Step, Reset
  - Debug / status messages

- **Extras**
  - Drag-and-drop commands (basic)
  - Responsive design for all devices
  - Smooth animations and pastel cyber aesthetic
